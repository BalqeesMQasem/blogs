Simple Python API


Create a virtual environment by running the following command:

For Windows:
python -m venv venv

For macOS/Linux:
python3 -m venv venv


Activate the virtual environment:

For Windows:
venv\Scripts\activate

For macOS/Linux:
source venv/bin/activate
